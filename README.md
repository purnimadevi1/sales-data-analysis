# Sales Data Analysis

This project analyzes retail sales data to identify trends and insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis
- Monthly sales trends
- Sales by region
- Sales by category
- Top products and cities

## Key Insights
- West region generated the highest revenue
- Technology category dominates sales
- A few cities contribute most of the revenue

## Dataset
Superstore Sales Dataset
